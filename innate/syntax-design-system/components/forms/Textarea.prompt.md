Use for free-form multi-line input — application essays, feedback, notes. Same label / hint / error contract as `Input`.

```jsx
<Textarea
  label="Why this program?"
  hint="A few sentences is plenty."
  rows={5}
/>
<Textarea label="Notes" autoResize defaultValue="Grows as you type." />
```

`autoResize` disables the manual drag handle and grows to fit.
