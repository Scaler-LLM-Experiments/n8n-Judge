Rules
Four things that keep a mixed-icon page looking consistent.
Do
arrow_forwardSee the curriculum
Inline-match icons to the text size — 20px icon next to 14–16px copy.
Don't
arrow_forwardSee the curriculum
Oversize icons inside body copy — it breaks rhythm and reads as a decoration, not a cue.
Do
shieldboltauto_awesome
Mix glyphs within the Sharp pack only — weight 400, grade 0, opsz 24.
Don't
shield bolt auto_awesome
Mix weights or fills within one layout — the visual rhythm falls apart.
Implementation: load the font once; render glyphs with <span class="ico">glyph_name</span>. For filled variants add .ico--fill. The variable axes give you fine-grained control when you need it — don't reach for other packs.
