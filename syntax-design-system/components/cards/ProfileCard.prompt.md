People-led card for landing pages — mentors, alumni, testimonials. Portrait-forward so the page feels human.

```jsx
<ProfileCard
  photo="https://images.unsplash.com/photo-1500648767791-00dcc994a43e?w=600"
  quote="Six months in, I shipped my first ML system at work."
  name="Rohan Mehta"
  role="SDE II"
  company="Razorpay"
  stat="13× salary jump"
/>
<ProfileCard layout="side" photo="…" name="Priya R." role="Alum, Cohort 12" />
```

`layout="top"` (default) for a grid of testimonials; `layout="side"` for compact rows. Use real, cool-leaning portraits.
