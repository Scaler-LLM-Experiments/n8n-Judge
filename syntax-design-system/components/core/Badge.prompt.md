Use to label status, category, or count. Tone communicates meaning; reserve `solid` for the one badge you want to dominate.

```jsx
<Badge tone="success">Enrolled</Badge>
<Badge tone="warning" icon={<Clock size={12} />}>Closing soon</Badge>
<Badge solid>New</Badge>
```

Tones: `neutral | info | success | warning | danger`. No rounded corners.
